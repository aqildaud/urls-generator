# EDM URL Generator

A simple tool I built to avoid the hassle of manually updating URLs for bulk uploaded files (images/files).

## What it does

When I upload a large number of files, I usually need to convert local file paths into live URLs one by one. This gets repetitive and time-consuming.

This tool fixes that by letting me:

- Paste multiple file paths at once
- Provide a base URL
- Generate all final live URLs instantly

## Why I built it

I was too lazy to keep updating URLs manually for every single file during bulk uploads. So I made this small utility to automate it in one go.
